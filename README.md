# Python Scientific Calculator

A clean, desktop-based scientific calculator built entirely in Python. It evaluates standard mathematical expressions, advanced trigonometric functions, and includes full keyboard support for rapid calculations.

## Features

* **Basic & Advanced Math:** Handles standard arithmetic alongside exponents, square roots, and logarithms.
* **Trigonometry & Constants:** Built-in support for `sin()`, `cos()`, `tan()`, as well as mathematical constants like pi and e.
* **Keyboard Integration:** Type expressions directly using your keyboard. Press `Enter` to calculate and `Backspace` to delete.
* **Error Handling:** Automatically catches invalid expressions or syntax errors (like division by zero) and safely displays an "Error" message without crashing the program.
* **Clean GUI:** A fixed-size, grid-based Tkinter interface with an easy-to-read, right-justified display.

## Getting Started

Follow these steps to run the calculator on your local machine.

### Prerequisites

You only need Python installed on your computer. This script relies exclusively on Python's standard libraries (`tkinter` and `math`), so there is absolutely no need to run any `pip install` commands.

### Running the App

1. Clone the repository to your local machine:
```bash
git clone [https://github.com/arishmukarram/Calculator.git](https://github.com/arishmukarram/Calculator.git)
```
Navigate into the project folder.

Launch the application:

```Bash
python calculator.py
```
